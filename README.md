# gulp_for_buge
## 中大型项目下的gulp自动化工作流
### ***使用方法:
### 1.自动更新页面  gulp server
###2.打包压缩   gulp build
### 3.生成zip gulp zip
###***功能: 
### 1.压缩js/css/图片
### 2.合并文件
### 3.重命名文件
### 4.编译ES6语法
###5.自动添加css前缀
### 6.自动刷新页面
### 7.生成雪碧图
### 8.删除目录或文件
### 9.编译less
### 10.生成压缩文件
### ***注意
### 不使用babel编译时gulp-minJs不支持es6压缩，会报错
### require.js系统不能用babel编译
### 若对文件改变后浏览器没有实时更新,此时手动刷新即可(该问题是浏览器缓存造成的)
### 无法同步对文件的删除操作
